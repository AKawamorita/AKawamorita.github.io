    Instalação do ambiente virtual

    1. Criar ambiente virtual

    Windows PowerShell

    python -m venv .venv
.\.venv\Scripts\Activate.ps1

    Linux/macOS

    python -m venv .venv
source .venv/bin/activate

    2. Atualizar pip

    python -m pip install --upgrade pip

    3. Instalar dependências

    pip install -r requirements.txt

    4. Configurar variáveis de ambiente

    Copie o arquivo de exemplo:

    cp .env.example .env

    No Windows PowerShell:

    copy .env.example .env

    Edite o .env:

    OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_LLM_MODEL=gemma4:e4b
OLLAMA_EMBEDDING_MODEL=nomic-embed-text

    O valor de OLLAMA_LLM_MODEL deve ser exatamente igual ao nome exibido por:

    ollama list

    5. Validar Ollama

    Teste se o Ollama está ativo:

    ollama list

    Baixe o modelo de embeddings:

    ollama pull nomic-embed-text

    6. Rodar ingestão

    python -m src.ingest --reset

    7. Fazer uma consulta

    python -m src.rag_chain "Qual é o fornecedor da invoice INV-1001?"

    8. Rodar interface

    streamlit run src/app.py

    9. Rodar documentação

    mkdocs serve
